File information
-----------------

e330_UIUC.dat		Original airfoil DAT file from UIUC.
e330.dat		Modified DAT file from Todd Colten. The file has been modified to suit AVL's syntax.
e330.m			M-file that reads data from e330.dat and can be used to plot airfoil in MATLAB.
e330_alphaSweep.txt	Output of alpha sweep conducted using XFOIL.
e330.af			AF file that can be read by Vehicle Sketch Pad.
